# Lab06 - Bai 3

Project mẫu cho **Lab 06 – Bài 3**.

## Chạy test (VS Code / Terminal)
- Mở thư mục project bằng VS Code (cài Extension Pack for Java)
- Chạy lệnh:

```bash
mvn test
```

## Ghi chú
- JUnit đang dùng: **JUnit 4**
- Bạn có thể sửa package name / code theo yêu cầu giảng viên.


> Lưu ý: test demo `demoErrorCollector_willFail()` đang được `@Ignore` để `mvn test` chạy xanh. Bỏ `@Ignore` để xem ErrorCollector lưu vết nhiều lỗi như trong lab.
